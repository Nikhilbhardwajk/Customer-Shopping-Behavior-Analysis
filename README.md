# Customer-Shopping-Behavior-Analysis
Transactional retail data analysis with feature engineering, SQL business queries, and interactive Power BI dashboard development.
# 1. Project Overview

This project analyzes customer shopping behavior using transactional retail data to uncover insights related to spending patterns, customer segmentation, product performance, and promotion effectiveness.

The objective was to transform raw purchase data into actionable business intelligence that can support strategic decisions in marketing, pricing, customer retention, and revenue optimization.

The analysis was performed using an end-to-end analytics workflow including data cleaning in Python, business analysis using SQL, and interactive dashboard development in Power BI.

# 2. Business Problems Solved

Identified high revenue customer segments

Evaluated whether discounts increase purchase value

Analyzed subscription impact on customer lifetime value

Detected products heavily dependent on discounts

Compared spending behaviour across age groups and genders

Measured the effect of shipping preference on purchase intent

# 3. Dataset Information

Total Transactions: 3,900

Total Features: 18

Categories: Clothing, Footwear, Accessories, Outerwear

Includes demographic, transactional, and behavioral variables

# 4. Data Processing & Feature Engineering

Handled missing review ratings using category-wise median imputation

Created new analytical features:

Age Groups

Customer Segments (New / Returning / Loyal)

Purchase Frequency Indicators

Removed redundant promotional variables to improve signal clarity

Loaded cleaned data into PostgreSQL for scalable SQL analysis

# 5. Key Insights

Young Adult customers generated the highest revenue contribution

Customers choosing Express Shipping showed slightly higher spending behaviour

Some products showed ~50% dependency on discounts, indicating pricing sensitivity

A large portion of customers belonged to the Loyal segment, indicating strong retention

Non-subscribers contributed more total revenue due to higher population size

# 6. Dashboard

An interactive Power BI dashboard was developed to visualize:

Revenue distribution

Customer segmentation

Category performance

Subscription analysis

Age-group sales trends

# 7. Business Recommendations

Improve subscription value proposition to increase Customer Lifetime Value

Optimize discount strategies to protect profit margins

Focus marketing campaigns on high-revenue demographic segments

Promote top-rated and best-selling products to improve conversion rates

Strengthen loyalty programs to increase repeat purchase behaviour
